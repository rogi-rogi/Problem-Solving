# BOJ (Baekjoon Online Judge)
지금까지 푼 문제의 코드들이 잠들어 있는 곳입니다.

많이는 풀었지만 잘하지는 못합니다!

필요할 때 변한다는 말이 있지요.

어느 순간 이전에 만든 코드를 다시 읽기가  ~~오직 코딩의 신만 읽을수 있을정도로~~ 어려워 가독성에 노력 3방울 정도 첨가했습니다.

Q : 어느 순간이 언제인지 어떻게 알죠?

A : 제가 군대 가기 전 까지 입니다.
